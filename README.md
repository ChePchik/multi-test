# multi-test

## Инструкция:

Client:

1.1. `cd client`

1.2. `npm i` - install

1.3. `npm run build` - собрать приложение(появиться папка build)

Server:

2.1. `cd server`

2.2. Переименовать .env-example в .env

2.3. В .env вставить свои данные, там всё предельно ясно

2.4. `npm i` - install

2.5. `npm run server`- run api server nodemon

2.6. go to `htttp://localhost:3003` - server

Преждче чем всё делать, [установить nodejs](https://nodejs.org/ru/)

И включить БД
