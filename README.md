# multi-test

Server:

- `cd server`
- переименовать .env-example в .env и вставить свои данные
- `npm i` - install
- `npm run server`- run api server nodemon
- go to `htttp://localhost:3003` - server

Client:

- `cd client`
- `npm i` - install
- `npm run start` - development client
- go to `htttp://localhost:3000` - client
